서버사이드 렌더링 
웹브라우저에서 주문내역을 요청한다.
그러면 서버는 주무DB 조회하고, html 동적으로 다 생성후 최종적으로 html 서버에서 다 생성하낟.
html 화면을 다 렌더링해서 http 응답에 코드를 다 실어서 보낸다
html 만드는걸 서버에서 다 끝내고 웹 브라우저는 렌더링만한다.
